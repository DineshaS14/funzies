# 💪 Gym Buddy Quiz 💘

A playful, romantic, and interactive mini quiz built with HTML, CSS, and JavaScript — asking the most important question of all:  
**"So ... would you not be my gym buddy?"** 😁💪

### 🎯 Features

- Animated background using a looping GIF
- Cute heartbeat animation on the quiz box
- Toast notification when an option is selected
- Marquee hint about free matcha protein 🍵✨
- Responsive and centered layout
